# ColorSurvivor

## 项目结构
```
ColorSurvivor
├─addons            # 插件
├─characters        # 角色
├─common            # 通用工具
├─components        # 通用组件
├─Levels            # 主场景
├─managers          # 全局类
├─pickable_items    # 可拾取道具
├─ui                # 用户界面
└─weapons           # 武器
```
